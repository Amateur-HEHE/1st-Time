Introduction to Pandas

Pandas is the foundational library for data manipulation and analysis in Python. Built on top of NumPy, it provides powerful, expressive data structures.

1What is Pandas?

🐼 Pandas

An open-source Python library providing data structures (Series & DataFrame) and tools for data analysis.

📅 Created in 2008

Wes McKinney developed pandas for financial data analysis at AQR Capital Management.

🏗️ Built on NumPy

Pandas uses NumPy arrays internally, gaining speed and rich mathematical operations.

2Installing Pandas
bash
pip install pandas
pip install numpy  # dependency
3Importing Pandas
python
import pandas as pd     # standard alias
import numpy as np      # usually imported together

print(pd.__version__)   # check version
💡
By convention, pandas is always imported as pd. Using this alias makes your code consistent with thousands of tutorials and projects.

📌 Chapter 5

Indexing & Selection

Master loc, iloc, and column selection — the most critical pandas skills.

🎯
loc = Label-based (use column names and row labels). iloc = Integer position-based (use row/column numbers starting at 0).
1Column Selection
python
df = pd.DataFrame({
    'A': [1,2,3], 'B': [4,5,6], 'C': [7,8,9]
})

# Single column → Series
df['A']            # dot notation also works: df.A

# Multiple columns → DataFrame
df[['A', 'C']]

# Add new column
df['D'] = df['A'] + df['B']  # element-wise addition
df['E'] = 'constant'          # broadcast scalar

# Delete column
df.drop('E', axis=1, inplace=True)  # or del df['E']
2loc — Label Based
python
df = pd.DataFrame({
    'Name': ['Alice','Bob','Charlie'],
    'Age':  [25, 30, 35],
    'City': ['Delhi','Mumbai','Chennai']
}, index=['a','b','c'])

df.loc['a']           # row with label 'a'
df.loc['a':'b']       # rows 'a' to 'b' (INCLUSIVE!)
df.loc['b', 'Name']   # cell at row 'b', col 'Name'
df.loc[:, 'Age']      # all rows, 'Age' column
df.loc['a':'b', ['Name','Age']]  # slice rows + cols
df.loc[df['Age'] > 25]  # boolean indexing
3iloc — Integer Based
python
df.iloc[0]            # first row
df.iloc[-1]           # last row
df.iloc[0:2]          # rows 0,1 (END exclusive!)
df.iloc[1, 2]         # row 1, col 2 (cell)
df.iloc[:, 1]         # all rows, col index 1
df.iloc[0:2, 1:3]     # rows 0-1, cols 1-2
df.iloc[[0,2], [0,1]] # specific rows + cols
⚠️
loc slice is inclusive on both ends. iloc slice is exclusive on the end — like Python lists. This is a common source of bugs!

✂️ Chapter 6

Slicing

Extract subsets of DataFrames using powerful slicing syntax and boolean masks.

1Row Slicing
python
df[1:4]      # rows 1,2,3 (integer slice, exclusive end)
df[:3]       # first 3 rows
df[::2]      # every 2nd row
df[::-1]     # reverse all rows
2Boolean Masking
python
# Single condition
df[df['Age'] > 25]

# Multiple conditions (use & | ~, NOT and/or/not)
df[(df['Age'] > 25) & (df['City'] == 'Delhi')]
df[(df['Salary'] > 50000) | (df['Age'] < 30)]
df[~(df['Dept'] == 'HR')]   # NOT HR

# isin — membership test
df[df['City'].isin(['Delhi', 'Mumbai'])]

# between — range check
df[df['Age'].between(25, 35)]

# String methods
df[df['Name'].str.startswith('A')]
df[df['Name'].str.contains('li', case=False)]
💡
Always use &, |, ~ for element-wise boolean ops — never Python's and / or / not with pandas!

Data Cleaning

Real data is messy. Learn to rename, retype, remove duplicates, and standardize your data.

1Rename Columns
python
# Rename specific columns
df.rename(columns={'old_name': 'new_name',
                   'col2': 'better_name'}, inplace=True)

# Rename all columns (must match count)
df.columns = ['A', 'B', 'C', 'D']

# Clean: lowercase all, replace spaces
df.columns = df.columns.str.lower().str.replace(' ', '_')
2Change Data Types
python
df['Age'] = df['Age'].astype(int)
df['Price'] = df['Price'].astype(float)
df['Date'] = pd.to_datetime(df['Date'])
df['Category'] = df['Category'].astype('category')  # memory efficient
df['Flag'] = df['Flag'].astype(bool)

# Safe conversion with error handling
df['col'] = pd.to_numeric(df['col'], errors='coerce')  # bad → NaN
3Remove Duplicates
python
df.duplicated()                          # bool mask of dupes
df.duplicated().sum()                    # count of dupes
df.drop_duplicates(inplace=True)         # remove all dupes
df.drop_duplicates(subset=['Name'])      # dupes by column
df.drop_duplicates(keep='last')          # keep last occurrence
4String Cleaning
python
df['Name'] = df['Name'].str.strip()      # remove whitespace
df['Name'] = df['Name'].str.lower()      # lowercase
df['Name'] = df['Name'].str.upper()      # uppercase
df['Name'] = df['Name'].str.title()      # Title Case
df['Name'] = df['Name'].str.replace(' ', '_')
df['Name'] = df['Name'].str.replace(r'\s+', ' ', regex=True)

# Extract with regex
df['code'] = df['field'].str.extract(r'([A-Z]{2}\d+)')

❓ Chapter 8

Handling Missing Data

Detect, handle, and impute missing values — a crucial step in every data pipeline.

1Detecting Missing Values
python
df.isnull()              # True where NaN
df.notnull()             # True where not NaN
df.isnull().sum()        # count NaN per column
df.isnull().sum() / len(df) * 100  # percentage missing

# Visualize missing pattern
df.isnull().sum().plot(kind='bar')  # quick bar chart
2Dropping Missing Values
python
df.dropna()                        # drop rows with ANY NaN
df.dropna(axis=1)                  # drop columns with ANY NaN
df.dropna(how='all')               # drop only if ALL values NaN
df.dropna(thresh=3)                # keep rows with at least 3 non-NaN
df.dropna(subset=['Age','Salary']) # only check these cols
3Filling Missing Values
python
df.fillna(0)                        # fill all NaN with 0
df['Age'].fillna(df['Age'].mean())  # fill with mean
df['City'].fillna('Unknown')        # fill with string
df.fillna(method='ffill')           # forward fill (propagate last value)
df.fillna(method='bfill')           # backward fill
df['Score'].fillna(df['Score'].median(), inplace=True)

# Fill different columns differently
df.fillna({'Age': df['Age'].mean(),
           'City': 'Unknown', 'Salary': 0})

Operations on DataFrame

Arithmetic, comparison, apply, and vectorized operations on DataFrames.

🗂️ Chapter 11

GroupBy & Aggregation

Split data into groups, apply functions, and combine results — the powerful Split-Apply-Combine pattern.

🧩
Split-Apply-Combine: Split data by group → Apply a function (sum, mean, etc.) → Combine results into a new DataFrame.
python
df = pd.DataFrame({
    'Dept': ['HR','Tech','HR','Tech','Finance'],
    'Name': ['Alice','Bob','Charlie','Diana','Eve'],
    'Salary': [50000, 90000, 55000, 85000, 70000],
    'Score': [80, 95, 75, 90, 88]
})

# Basic groupby
df.groupby('Dept')['Salary'].mean()       # avg salary per dept
df.groupby('Dept')['Salary'].sum()        # total salary per dept
df.groupby('Dept').size()                 # count per dept

# Multiple columns & aggregations
df.groupby('Dept')[['Salary','Score']].mean()

# agg — multiple functions at once
df.groupby('Dept')['Salary'].agg(['mean','std','min','max'])

# Named aggregation (pandas 0.25+)
df.groupby('Dept').agg(
    avg_salary=('Salary', 'mean'),
    total_salary=('Salary', 'sum'),
    headcount=('Name', 'count')
)

# Transform — returns same-shape df (keeps index)
df['dept_avg'] = df.groupby('Dept')['Salary'].transform('mean')

# Filter — keep groups matching condition
df.groupby('Dept').filter(lambda x: x['Salary'].mean() > 70000)

🔗 Chapter 12

Merge, Join & Concatenate

Combine multiple DataFrames — like SQL JOINs but in Python.

1pd.merge (SQL-style JOIN)
python
employees = pd.DataFrame({'EmpID':[1,2,3], 'Name':['Alice','Bob','Charlie']})
salaries  = pd.DataFrame({'EmpID':[1,2,4], 'Salary':[50000,60000,70000]})

pd.merge(employees, salaries, on='EmpID', how='inner')  # common rows
pd.merge(employees, salaries, on='EmpID', how='left')   # all left + matching right
pd.merge(employees, salaries, on='EmpID', how='right')  # all right + matching left
pd.merge(employees, salaries, on='EmpID', how='outer')  # all rows, NaN where missing

# Merge on multiple columns
pd.merge(df1, df2, on=['Key1','Key2'], how='inner')

# Different column names
pd.merge(df1, df2, left_on='emp_id', right_on='employee_id')
2pd.concat (Stack DataFrames)
python
df1 = pd.DataFrame({'A': [1,2], 'B': [3,4]})
df2 = pd.DataFrame({'A': [5,6], 'B': [7,8]})

pd.concat([df1, df2])               # stack vertically (axis=0)
pd.concat([df1, df2], ignore_index=True)  # reset index
pd.concat([df1, df2], axis=1)       # stack horizontally

# Stack multiple dataframes
all_dfs = [df_jan, df_feb, df_mar]
combined = pd.concat(all_dfs, ignore_index=True)
howSQL equivalentResult
innerINNER JOINOnly matching rows
leftLEFT JOINAll left + matching right
rightRIGHT JOINAll right + matching left
outerFULL OUTER JOINAll rows from both
🔃 Chapter 13

Sorting & Filtering

Sort by values or index, filter rows with conditions, and rank your data.

1Sorting
python
df.sort_values('Salary')                           # ascending default
df.sort_values('Salary', ascending=False)          # descending
df.sort_values(['Dept','Salary'], ascending=[True,False])  # multi-col
df.sort_values('Score', na_position='last')        # NaN at end
df.sort_index()                                    # sort by index
df.sort_index(ascending=False)                     # reverse index

# Rank
df['Salary_rank'] = df['Salary'].rank(ascending=False, method='dense')

# nlargest / nsmallest
df.nlargest(5, 'Salary')      # top 5 by salary
df.nsmallest(3, 'Age')        # bottom 3 by age
2Filtering & Masking
python
# where — keeps structure, replaces False with NaN
df['Salary'].where(df['Salary'] > 50000)

# mask — opposite of where
df['Score'].mask(df['Score'] < 60, 0)  # below 60 → 0

# clip — bound values to range
df['Score'].clip(lower=0, upper=100)

# between
df[df['Age'].between(25, 35, inclusive='both')]

# query (SQL-like)
df.query('Salary > 60000 and Dept == "Tech"')

python
import pandas as pd

# Load sales data

print(f"Top Product: {top_product}")

monthly = df.groupby('Month')['Revenue'].sum().reset_index()
monthly.sort_values('Revenue', ascending=False)
